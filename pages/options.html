JSHint Options

This page's content is sourced from the JSHint project repository. If you spot an error, please open an issue or (better yet) make a pull request!

Default Behavior

JSHint will impose some constraints by default. If you would like to make JSHint less strict, you can do so by enabling specific Relaxing options (see the full list of Relaxing options). If you would like to make JSHint more strict, you can do so by enabling specific Enforcing options (see full list of Enforcing options). JSHint may be configured with whatever combination of these options you like.
